4. Instruções de Uso

  • Insira a senha na seção "Acesso Restrito" e clique em "Desbloquear Acesso" para habilitar a adição e exclusão de despesas.
  • Preencha todos os campos do formulário "Controle de Despesas" para cada nova aquisição ou pagamento.
  • Clique no botão "Adicionar Despesa" para registrar o item na tabela.
  • O "Total Já Gasto" e o "Saldo Disponível" no "Resumo Financeiro" são atualizados automaticamente.
  • Verifique os dados inseridos com atenção para garantir a precisão das informações.
  • Para remover uma despesa, clique no botão "Excluir" na respectiva linha da tabela (requer acesso desbloqueado).
  • Esta planilha visa garantir o controle financeiro e a transparência na prestação de contas dos recursos.
